### Get the joke

**Description:**
A simple web app that fetches and displays random jokes from a joke API.

**Features:**

1. **Fetch Joke Button:**
   - Users can click a button to fetch and display a random joke.

2. **Joke Display:**
   - Show the fetched joke on the webpage.

**Technologies:**

- **Frontend:**
  - React + Vite.

- **API:**
  - Use a Joke API (for example, the JokeAPI: https://v2.jokeapi.dev/).

**Learning Outcomes:**
- Setting up a react project with Vite.
- Making API requests using JS.
- Displaying dynamic content on a webpage using React.
- Handling API responses.

**Not yet implemented bonus features:**

- Interface to fetch a joke from a specific category (e.g., programming jokes).
- Include a simple animation or transition when displaying a new joke.
